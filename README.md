# Dalai Strong (React + Vite + Tailwind)

## Scripts
- `npm install`
- `npm run dev` (localhost)
- `npm run build` (producción)
- `npm run preview` (previsualizar build)

## Subir a GitHub (GitHub Desktop, Windows)
1. Abrí **GitHub Desktop** > **File > Add local repository** > seleccioná esta carpeta.
2. **Commit** inicial (mensaje: "init Dalai Strong") y **Publish repository**.
3. Luego cada cambio: **Commit to main** > **Push origin**.

> Si preferís CLI: `git init && git add . && git commit -m "init" && git branch -M main && git remote add origin <URL> && git push -u origin main`
